## âš™ï¸ Getting Started

Follow these steps to get the application up and running locally.

### Prerequisites

Before you begin, ensure you have the following installed on your system:

  * **Java Development Kit (JDK) 17+** (Required for the Spring Boot backend)
  * **Maven** or **Gradle** (For building the Spring Boot backend)
  * **Node.js & npm** (Required for the Angular frontend)
  * **MySQL Server** (Required for the database)

### 1\. Database Setup

1.  Navigate to the `database/` directory.

2.  Execute the provided SQL script(s) against your local MySQL instance to create the necessary schema and tables.

    ```bash
    # Example command (may vary based on your setup)
    mysql -u [USER] -p < database/[your_script_name].sql
    ```

### 2\. Backend Installation and Run

1.  Navigate to the `backend/` directory.

2.  Build the Spring Boot application (e.g., using Maven):

    ```bash
    cd backend
    ./mvnw install
    ```

3.  Ensure your **database connection details** (URL, username, password) are correctly configured in the `application.properties` or `application.yml` file within the backend source code.

4.  Run the application:

    ```bash
    ./mvnw spring-boot:run
    ```

    The backend API should now be running, typically on `http://localhost:8080`.

### 3\. Frontend Installation and Run

1.  Navigate to the `frontend/` directory.

2.  Install the required Node.js dependencies:

    ```bash
    cd frontend
    npm install
    ```

3.  Start the Angular development server:

    ```bash
    npm start
    ```

    The frontend application will compile and launch in your browser, typically on `http://localhost:4200`.
